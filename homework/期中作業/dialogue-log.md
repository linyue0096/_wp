# 俄羅斯方塊遊戲開發 — 對話紀錄

## 1. 初次建置：基本俄羅斯方塊遊戲
- 建立 `index.html`，單一 HTML 包含完整遊戲
- 標準 10×20 棋盤、7 種經典方塊（I, O, T, S, Z, J, L）
- 鍵盤控制：方向鍵移動/旋轉、空格直接落下
- 分數、等級、消行數顯示
- 下一塊預覽
- 等級越高掉落越快（`800 - (等級-1)*70`，最快 100ms）
- 觸控按鈕支援手機操作

## 2. 加入暫停／重新開始／最高分
- 新增暫停/繼續按鈕，按 `P` 鍵切換
- 新增獨立的重新開始按鈕（非僅 Game Over 時）
- 使用 `localStorage` 儲存最高分，金色顯示
- 遊戲說明面板（操作提示）

## 3. 三大遊戲模式（重構）
- **單人模式**：經典玩法，附最高分紀錄
- **人機對戰（AI）**：左右雙畫面，AI 對手使用啟發式演算法
- **好友對戰**：6 位數房號配對，WebSocket 即時同步
- 新增 `server.py`（Python WebSocket 伺服器，使用 `websockets` 套件）
- 選單系統：主選單 → 模式選擇
- 房間系統：建立房間／加入房間

## 4. AI 難度分級＋輔助線＋方格背景
- **三級難度**：
  - 簡單：高隨機性、偶爾故意放錯、基礎速度 900ms
  - 中級：少量隨機、速度 700ms
  - 困難：近乎完美、極低隨機、速度 400ms
- **掉落輔助線**：半透明框線顯示落點位置，按 `G` 切換
- **方格背景**：棋盤改為網格線樣式

## 5. Bug 修復
- 單人模式遊玩後無法切換到人機對戰選單
  - 原因：`backToMenu()` 未移除 `menuScreen` 的 `hidden` class
  - 修復：加入 `menuScreen.classList.remove('hidden')`

## 6. 速度上限調整
- 原版速度無限加速到 80ms（約 Lv13），Lv10 已非人類可玩
- 新增 `calcInterval(base, level)` 統一管理速度公式
- **Lv1~8**：維持每級 -60ms 遞減，最快為 Lv8
- **Lv9~20**：每兩級為一組，從 350ms 開始每組 -30ms
  - Lv9~10: 350ms, Lv11~12: 320ms, ..., Lv19~20: **200ms（上限）**
- AI 對手使用各自的 Lv8 速度 × 倍率遞減（每兩級 ×1.5 倍）
- 升級門檻：每 10 行 → **每 8 行**升一級（加快 0.2 倍）

## 7. 7-Bag 方塊隨機系統
- 取代純隨機 `randType()`，改用 7-bag 洗牌發牌
- 每袋含 1~7 號方塊各一，Fisher-Yates 洗牌後依序出牌
- 袋邊界檢查，避免同方塊連續出現
- 保證每種方塊出現率 **1/7**（滿足 ≥ 1/15 要求）
- 自然防止同方塊連續 3 次

## 8. 開發紀錄顯示
- 選單中加入「📜 開發紀錄」按鈕
- 彈出視窗顯示完整開發歷程與檔案結構
- 支援關閉按鈕與點擊外部關閉

## 9. 公開隧道服務
- 使用 localhost.run 建立 SSH 反向隧道
- 不需同 Wi-Fi 即可從外部連線遊玩
- 網址格式：`https://<隨機代碼>.lhr.life`
