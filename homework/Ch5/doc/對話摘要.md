OpenCode 網誌專案對話摘要
=============================

日期：2024年5月

專案目標：建立類似 Threads 風格的社群網誌後端系統

【技術堆疊】
- 後端：Node.js + Express
- 資料庫：SQLite3
- 前端：HTML + CSS + JavaScript (Threads 風格)
- 其他套件：bcryptjs, express-session, marked

【建立的核心檔案】

1. /blog/package.json
   - 進入點：app.js
   - 相依套件：express, sqlite3, bcryptjs, express-session, marked

2. /blog/database.js
   - SQLite 連線至 ./blog.db
   - 資料表：users, posts, likes, follows
   - 包含按讚與追蹤功能

3. /blog/app.js
   - Express 伺服器，監聽 port 3000
   - API 端點：
     * POST /register - 用戶註冊
     * POST /login - 用戶登入
     * POST /logout - 登出
     * GET /me - 取得當前用戶資訊
     * PUT /me - 更新個人資料
     * GET /users/:id - 取得用戶資料
     * POST /follow/:id - 追蹤用戶
     * DELETE /follow/:id - 取消追蹤
     * GET /feed - 追蹤動態
     * GET /posts - 公共貼文
     * GET /users/:id/posts - 用戶貼文
     * POST /posts - 發布貼文
     * DELETE /posts/:id - 刪除貼文
     * POST /posts/:id/like - 按讚
     * DELETE /posts/:id/like - 取消按讚

4. /blog/public/index.html
   - Threads 風格前端界面
   - 三欄式布局：左側導航、中央 feed、右側用戶資料
   - 功能：
     * 登入/註冊/登出
     * 發布貼文
     * 顯示公共貼文區（探索）
     * 顯示追蹤動態
     * 按讚功能
     * 追蹤/取消追蹤
     * 點擊作者頭像/名稱進入個人版面
     * 顯示用戶資料（貼文數、追蹤數、粉絲數）

【功能特點】
- 密碼使用 bcrypt 加密儲存
- Session 追蹤登入狀態
- 顯示用戶頭像（使用 DiceBear API）
- 支援追蹤/粉絲系統（類似 Threads）
- 公共貼文區顯示所有用戶貼文
- 個人貼文會出現在公共區
